How we compute each number

1,000 hand-built shopper agents walk the floor for 12 hours under your baseline operating policy. Each agent has a backstory, queue tolerance, dwell time, basket-size target, and zone preferences — all reproducible from a single RNG seed. Same seed, baseline vs optimized differ only because of policy, not luck.

What we will NOT do

Every claimed lift flows through a causal chain — policy change → environment change → agent reacts → revenue moves. No boost-multiplier knobs, no forced conversion rates, no fiat ATV bumps. If a finding cannot be traced back to a specific agent behaviour, it is not surfaced.
